COA: Logical AND Gate

The COA (Computer Organization and Architecture) project focuses on the implementation and understanding of logical AND gates in digital circuits. A logical AND gate is a fundamental component that outputs 1 only when both input signals are 1; otherwise, it outputs 0. This project explores the principles behind AND gates and their applications in digital electronics.


DSA: Singly Linked List

The DSA (Data Structures and Algorithms) project involves the implementation and exploration of singly linked lists. A singly linked list is a fundamental data structure consisting of nodes where each node contains data and a reference to the next node. This project delves into the operations and applications of singly linked lists, such as insertion, deletion, traversal, and searching, providing insights into efficient data storage and manipulation techniques.
